Quests

Cubic Odyssey has two parallel quest tracks: a hand-authored main story (149 progression tasks driving milestones like GOT_WARP_CELL, FREED_QUEEN, REACHED_NEXUS) and a random-quest system that spawns side jobs from a small pool of types. Both are visible in the game's static configs.

Tips

  • Random quests give Trading XP on completion — even if the reward is small, they're a steady drip. Pick them up at every outpost you visit.
  • Pirate bounties stack with the pirate's drop table. See the Enemies page for what each pirate variant drops; if a bounty asks for a tier-3 pirate, the drop table is pirate_tier3.
  • Planet / system charts are passive Trading XP — you'd be visiting those systems anyway, the quest just pays you for it.
  • The main story is gated by progression flags. Don't rush past biome-discovery beats — some equip / craft tasks require you to have actually picked up the item, not just have the recipe.
