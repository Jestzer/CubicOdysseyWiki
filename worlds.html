World types

Cubic Odyssey ships 9 world types in data/configs/worldbiomes/. Each defines a set of climate biomes and pairs with a voxel-distribution file that controls which ores spawn at each depth. Most planets pick biomes procedurally (m_biome* as "random"), so the world type tells you what to expect more reliably than the planet name.
